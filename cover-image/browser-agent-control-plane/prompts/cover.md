# Cover prompt — Browser-agent control plane

Create a 16:9 editorial hero image for a BrowserMan Blog article titled:

"The browser-agent control plane"

Visual concept:
A sleek dark-mode command center for AI browser agents. A real browser window sits at the center, surrounded by subtle control-plane overlays: scoped session boundary, approval gate, audit trail, handoff marker, rollback/revoke path. The design should feel like professional developer infrastructure, not sci-fi fantasy.

Mood:
Calm, precise, secure, operator-minded, premium devtools.

Style:
Elegant digital editorial illustration, crisp UI abstraction, subtle glow, BrowserMan/devtools aesthetic. No cheesy robots, no humanoid AI, no generic stock art, no padlock cliché. Prefer abstract browser/session boundaries, thin lines, small status chips, terminal/control panels, and a visible but tasteful boundary around the browser session.

Composition:
- Wide 16:9 layout.
- Central browser window/control surface.
- Surrounding layers labeled visually or symbolically: Scope, Gate, Audit, Handoff, Revoke.
- Good negative space for social card cropping.
- High contrast but not noisy.

Palette:
Dark charcoal / near-black background, muted blue/cyan highlights, small warm amber approval accents, subtle off-white text/lines.

Text:
Minimal. If text is used, only short UI labels like "scope", "gate", "audit", "revoke". Do not render the full article title as text unless typography is clean and exact.

Avoid:
Robots, human faces, fantasy cyberpunk, excessive neon, locks as main metaphor, broken browser windows, cluttered dashboards, misspelled text.
